gradientBox

public static function gradientBox(graphics: Graphics,  
object: IValued,  
w: Number,  
h: Number,  
x: Number = 0,
y: Number = 0,
a: Number = 100,
prefix: String = ''):void

Draws a rounded, colored box with optional border.  The graphics object will be cleared before drawing begins.  The following keys should be in xml or object parameter:

  • angle: Number indicating direction of gradient in degrees
  • border: Number indicating border width in pixels
  • bordercolor: String containing six character hex color for border
  • color: String containing six character hex color for base
  • curve: Number indicating how much to curve edges in pixels
  • grad: Number indicating how much to gradate the color

Parameters

graphicsGraphics object to draw into
xmlXML object to look for attributes in
objectObject to look for keys in
wNumber indicating the horizontal size of box in pixels
hNumber indicating the vertical size of box in pixels
xNumber indicating horizontal offset of box in pixels
yNumber indicating vertical offset of box in pixels
aNumber indicating alpha transparency of box as percentage
